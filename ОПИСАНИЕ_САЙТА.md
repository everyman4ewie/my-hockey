## Общая структура

Платформа для хоккейных тренеров: план-конспекты тренировок, тактические доски, экспорт в PDF/Word.

---

## Маршруты


| Путь            | Страница                    | Кто видит                     |
| --------------- | --------------------------- | ----------------------------- |
| `/`             | Главная (Landing)           | Все                           |
| `/login`        | Вход                        | Все                           |
| `/register`     | Регистрация                 | Все                           |
| `/cabinet`      | Личный кабинет              | Пользователь (админ → /admin) |
| `/admin`        | Админ-панель                | Только админ                  |

| `/board`        | Новая тактическая доска     | Пользователь                  |
| `/board/:id`    | Редактировать доску         | Пользователь                  |
| `/payment/test` | Тестовая оплата             | Пользователь                  |

## Страницы

### Главная (/)

- Hero, О нас, Возможности, Цена, Контакты
- Контент редактируется в админке
- Кнопки: Войти / Личный кабинет

### Вход (/login)

- Логин (или email) + пароль
- Админ → /admin, пользователь → /cabinet

### Регистрация (/register)

- Email, пароль, капча
- Логин = часть email до @

### Личный кабинет (/cabinet)

- **Личные данные**: логин, имя, дата рождения, команда, фото, логотип команды, смена пароля
- **Мои план-конспекты**: список планов и досок, кнопки создания и редактирования
- **Тарифы**: текущий тариф, кнопка «Купить» → /payment/test

### Админ-панель (/admin)

- Дашборд, пользователи, план-конспекты, настройки, тарифы
- Назначение тарифов пользователям (в т.ч. Ultima)
- Редактор контента главной страницы

### План-конспект (/plan/new, /plan/:id)

- Название + список упражнений
- Каждое упражнение: canvas (хоккейная площадка) + заметки
- Экспорт в PDF и Word
- Логотип команды 40×40 в левом верхнем углу canvas

### Тактическая доска (/board, /board/:id)

- Рисование на хоккейной площадке (линии, стрелки, иконки игроков)
- Выбор зоны: полная площадка / половина (атака/оборона) / чистый фон
- Сохранение, скачивание PNG
- Логотип команды 40×40 в левом верхнем углу

### Оплата (/payment/test)

- Тестовая страница: выбор способа оплаты, ввод данных карты
- Параметры: tariffId, period (month/year)
- Пока без реальной оплаты

## Тарифы и лимиты


| Тариф     | Планов | PDF  | Word | PNG доски | Упражнений в плане      |
| --------- | ------ | ---- | ---- | --------- | ----------------------- |
| free      | 2      | 2    | 0    | 0         | 5                       |
| personal  | 20     | 20   | 10   | 60        | ∞                       |
| corporate | 2000   | 2000 | 2000 | 3000      | ∞                       |
| unlimited | ∞      | ∞    | ∞    | ∞         | ∞                       |
| ultima    | ∞      | ∞    | ∞    | ∞         | ∞ (только через админа) |

## Данные

- **data.json** — пользователи, план-конспекты, доски
- **admin.json** — профиль админа, контент лендинга
- **localStorage** — токен, пользователь, черновики

---

## API (кратко)

- `/api/auth/`* — регистрация, вход
- `/api/user/profile` — профиль, тариф, usage
- `/api/user/usage/check` — проверка лимита перед скачиванием
- `/api/plans/`*, `/api/boards/*` — CRUD план-конспектов и досок
- `/api/admin/*` — админские эндпоинты
- `/api/pages/landing` — контент главной
